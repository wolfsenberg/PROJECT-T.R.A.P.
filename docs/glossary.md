# Glossary

This glossary is manually curated and should stay aligned with source-backed notes.

## Content Terms

| Term | Meaning |
| --- | --- |
| Topic | One of the 7 subject areas in the reviewer. |
| Review Notes | Source-backed memorization material for a topic. |
| Exam Mode | Question-answer practice mode for a topic. |
| Table Notes | Notes formatted as Term, Definition, Remember, Common Mistake. |
| Common Mistake | A warning about common confusion or exam pitfalls. |
| Quick Drills | Short prompts students can use before taking the exam. |
| Source-backed draft | Content has topic-level sources but has not completed row-level audit. |
| Verified | Content has been checked against reliable sources. |
| Needs verification | Content should not be treated as final until checked. |
| Source Panel | The visible notes-page list of topic-level citations. |

## Product Terms

| Term | Meaning |
| --- | --- |
| Incoming Student | Student preparing to enter PUP CCIS. |
| Ladderized Student | Student preparing for a ladderized or qualifying assessment. |
| Retake | Starting a fresh attempt for a topic that was already submitted. |
| Attempt History | Saved record of completed full-exam results. |
| XP UI | Windows XP-inspired interface style used by the app. |
