# Product Requirements

## Purpose

The platform is only for PUP CCIS entrance, qualifying, ladderized, and mock exam review. It is not intended to become a general course portal.

## Users

- Incoming PUP CCIS students.
- Students preparing for ladderized or qualifying assessment.

## Scope

- The platform has 7 final review topics for now:
  - IT Fundamentals
  - Data Structures & Algorithms
  - Operating Systems
  - Discrete Structures
  - Data Communication & Networking
  - Web Development
  - Human-Computer Interaction
- Each topic has Review Notes and Exam Mode.
- Notes may go beyond current exam questions if the additional material fits the PUP CCIS exam-review context.
- Exams may be retaken.

## Review Notes

Notes must support both:

- Fast memorization through tables.
- Deeper reading through focused sections and quick drills.

The primary table format is:

| Term | Definition | Remember | Common Mistake |
| --- | --- | --- | --- |

Do not add an Example column yet because the UI is already dense.

## Exam Mode

- Answers must be shuffled so correct answers do not follow predictable letters.
- Results must remain visible until the user chooses to continue.
- Submitted topics can be retaken.
- If an audit finds an incorrect question, answer, or explanation, fix it.

## Verification

- Notes should show a visible status.
- Current statuses:
  - Verified
  - Source-backed draft
  - Needs verification
- Uncertain content should be marked as Needs verification instead of presented as final.

## User-Facing Language

Notes should be in English.
