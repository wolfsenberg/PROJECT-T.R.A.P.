# Source Policy

## Source Priority

Use sources in this order:

1. PUP or CCIS materials.
2. CHED or official Philippine law sources.
3. Official documentation and standards, such as RFC, W3C, MDN, PHP Manual, ISO, NIST.
4. Textbooks and open courses.
5. Secondary sources.

## PUP Alignment

When public PUP or CCIS material is unavailable, use reliable external sources but keep content aligned to the PUP CCIS exam-review purpose. Do not expand into unrelated professional certification material unless it supports the 7 topic areas.

## Citation Rule

- Source per topic is enough for the user-facing app.
- Internal audits can cite more specific sources per table, row, or exam item when needed.
- Visible citations should appear in the notes page source panel.

## Conflict Rule

If sources disagree, use what is actually correct according to the strongest available source. Prefer official standards and official legal text over secondary summaries.

If the current exam answer conflicts with verified source material, update the exam answer and explanation.

## Uncertain Content

Do not present uncertain content as verified. Mark it as Needs verification and record it in the internal content audit.
